# 1. 프로젝트 소개

---

💡 **개발동기** <br>
혼자 구매하기에는 **양**이나 **배송비**가 부담스러워서 선뜻 구매를 망설였던 제품을 
*같은 이유로 구매를 망설이고 있는* ***주변 이웃과 연결**해주면 어떨까?* 
하는 생각에서 시작된 **P2P 소분 거래 플랫폼**입니다.

▶️ **타겟 유저** <br>
1인 가구, 배송비 아끼고 싶은 사람 등

▶️ **기대 효과** <br>
**실시간 채팅**을 통해 구매 의사를 조율하여
사용자가 **제품을 보다 쉽고 합리적으로 구매**를 할 수 있는 **편의 제공**
